Installation - Windows

Download and Unpack

First download and install the message bus service.

To get the distributor go to the Releases page of the repo and download the distributor for your platform: typically. distributor-5.0.0-win10-x64.zip. Unzip the archive into a folder of your choice.

The examples in the documentation assume the distributer has been unpacked to C:\distributor to keep the paths simple.

To start the distrubter double-click on JetBlack.MessageBus.Distributor.exe to start it. The first time the service is run Windows will pop up a security alert indicating the service wants to access a network port. Click on "Allow access" to continue.

Add a Configuration File

Optionally create a file called appsettings.json in the installation folder using the default configuration given here.

Now create the file distributor.bat with the following contents.

C:\distributor\JetBlack.MessageBus.Distributor.exe C:\distributor\appsettings.json
