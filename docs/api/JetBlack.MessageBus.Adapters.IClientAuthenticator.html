Interface IClientAuthenticator

The abstract client authenticator.

Namespace: JetBlack.MessageBus.Adapters
Assembly: JetBlack.MessageBus.Adapters.dll
Syntax
public interface IClientAuthenticator
Remarks

In the case of a client which is acting as a proxy the authenticator is provided with the user the proxy is impersonating, and the host on which that client is running.
